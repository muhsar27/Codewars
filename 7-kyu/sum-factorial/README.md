<h2><a href=https://www.codewars.com/kata/56b0f6243196b9d42d000034/train/python target="_blank">Sum Factorial</a></h2><h3>7 kyu</h3><p>Factorials are often used in probability and are used as an introductory problem for looping constructs. In this kata you will be summing together multiple factorials.</p><p>Here are a few examples of factorials:</p><pre><code>4 Factorial = 4! = 4 * 3 * 2 * 1 = 246 Factorial = 6! = 6 * 5 * 4 * 3 * 2 * 1 = 720</code></pre><p>In this kata you will be given a list of values that you must first find the factorial, and then return their sum.</p><p>For example if you are passed the list <code>[4, 6]</code> the equivalent mathematical expression would be <code>4! + 6!</code> which would equal <code>744</code>.</p><p>Good Luck!</p><p>Note: Assume that all values in the list are positive integer values &gt; 0 and each value in the list is unique.</p><p>Also, you must write your own implementation of factorial, as you cannot use the built-in <code>math.factorial()</code> method.</p>
